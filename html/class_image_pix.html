Detailed Description

template<typename pixel>
class ImagePix< pixel >

Templated class to implement an image either of int or of ComplexNumber in the Fourrier domain It contains several constructor:

  • a default constructor
  • an empty constructor of the desired size
  • a constructor from the Pixelarray generated while reading an image (see the ImageTools.cpp file to know how images are read)
